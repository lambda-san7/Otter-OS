# Otter-OS
A Fake OS made in python

## Commands:

### example:
- command
    - method

- file
    - -**c**
        - used to **c**reate files
        - syntax: `file -c filename.type`
    - -**o**
        - used to **o**pen files
        - syntax: `file -o filename.type`
    - -**r**
        - used to **r**ead files
        - syntax: `file -r filename.type`
    - -**e**
        - used to **e**dit files
        - syntax: `file -e @lineNumber *content`
    - -**s**
        - used to display files in the **s**ystem 
        - syntax: `file -s`
